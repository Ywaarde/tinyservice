# Tinyservice

Prepare a virtual environment and install packages:

- Setup a virtual environment
``` 
    cd tinyservice
    python -m venv venv
```

- Activate the virtual environment
```    
    source venv/bin/activate
```
- Install packages 
```    
    pip install -r requirements.txt
```


To run the program
```    
    python wsgi.py
```

To run tests
```    
    python test_tinyservice.py
```
